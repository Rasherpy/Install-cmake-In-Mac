## 安装(以3.8.0为例)
* 1.解压：tar xvf cmake-3.8.0.tar
* 2.进入解压目录：cd cmake-3.8.0

进行如下操作:(选择适合自己的操作步骤)如果还没有安装CMake，源码树中提供了一个 bootstrap 脚本：
* 1. ./bootstrap
* 2. make
* 3. make install
